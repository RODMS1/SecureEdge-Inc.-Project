# SecureEdge Inc. Project

This repository contains the documentation and scripts for the SecureEdge Inc. project.

## Project Overview

This project aims to establish a secure and efficient IT infrastructure for SecureEdge Inc. The project includes the following key components:

### 1. Active Directory (AD) Setup

*   **Domain Controller:** A Domain Controller will be set up to manage the network.
*   **Employee Accounts:** Employee accounts will be created as per the organizational chart.
*   **Automated User Creation:** An automated script will be developed to streamline the creation of new user accounts.

### 2. Network Access Control (NAC)

*   **NAC Solution:** A NAC solution with a Captive Portal will be implemented using FreeRADIUS.
*   **AD Integration:** The NAC solution will be integrated with Active Directory for authentication.

### 3. Security Policies

*   **Password Policies:** Strong password policies will be configured to enforce strong authentication measures.
*   **Access Policies:** Access policies will be defined to ensure the principle of least privilege.

### 4. Workstation Standardization

*   **Standardized Environment:** A standardized workstation environment will be set up.
*   **Company-wide Wallpapers:** Company-wide wallpapers will be deployed to all workstations.

### 5. Custom Tooling

*   **Network Monitoring Tool:** A custom tool will be developed to identify and monitor devices on the network.

### 6. Internal Documentation Portal

*   **Web Server:** A web server will be deployed using Nginx to host internal documentation.
